# 连接 Connections

此界面能直观地查看当前所有连接的:

  • 类型
  • 入口
  • 连接建立的时间点

点击连接可查看更多信息:

  • 上传速度
  • 下载速度
  • 上传流量
  • 下载流量
  • 连接发起地址
  • 连接目标地址
  • 命中规则类型
  • 策略组链

也可掐断任意或全部连接.
